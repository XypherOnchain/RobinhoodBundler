Step 1 — Set up wallets

You need three kinds of wallets. Think of them like roles on a team.

Do this in order:
  1. Bank — import the wallet that holds your ETH
  2. Creator — create (or import) the wallet that will make the coin
  3. Helpers — create a few wallets that will buy the coin
  4. (Optional) Season helpers + use privacy hops when funding so they look less linked
  5. Go to Budget

A. Bank wallet (shared)

This is your treasury. It pays for everything. One bank is shared across all coin tabs.

B. Creator wallet (this coin only)

Creates the token and does the first small buy. Keep it separate from the bank and helpers.

C. Helper wallets (buyers)

These wallets buy after launch. More helpers = more buy transactions.

Optional: season helpers, export keys, sweep leftover cash

Step 2 — Set the budget

Decide how much ETH helpers will spend in total. We’ll split it across them (small buys first, larger later).

Do this:
  1. Leave token blank if you’ll create a new coin in Go live
  2. Or paste an existing coin address if you’re buying one already live
  3. Set total ETH and how many helpers
  4. Build plan → Save to wallets → Next

Step 3 — Send cash to helpers

Move ETH from your bank to each helper so they can buy. We send a little extra for network fees.

Do this:
  1. Make sure the budget was saved
  2. Preview → check the list looks right
  3. Send cash → wait until it finishes
  4. If some fail, hit Continue (skips wallets that already have cash)
  5. Go to Go live

Step 5 — Cash out

Watch the chart, then sell helper bags carefully. Prefer small sells so you don’t crash the price.

Do this:
  1. Load the chart (or open DexScreener)
  2. Refresh P&L to see what you hold
  3. Build recommendations for safe sells
  4. Sell a % from each wallet — don’t dump everything at once unless you’re done
