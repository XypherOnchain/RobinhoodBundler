Quick guide

Same flow as Stealth Bundler — four tabs, one job each.

  1. Wallets — import treasury, create deployer + sniper helpers
  2. Launch — plan budget → fund helpers → create coin → buy
  3. Trade — chart, sell, automations (patterns / bump / guards)
  4. Airdrop — send tokens to your tape or competitor buyers
  5. Overview — Mission Control + what to do next

Live automations need EVM_ARM_LIVE=true on the server plus ARM LIVE on the job.

Wallets

Treasury pays. Deployer creates the coin. Snipers buy.

Do this in order:
  1. Bank — import the wallet that holds your ETH
  2. Creator — create (or import) the wallet that will make the coin
  3. Helpers — create a few wallets that will buy the coin
  4. (Optional) Season helpers + use privacy hops when funding so they look less linked
  5. Go to Budget

A. Treasury (bank)

Shared across all coin tabs. Pays for funding + gas.

B. Deployer (creator)

Creates the token and does the first small buy. Keep it separate from the bank and helpers.

C. Snipers (buyers)

These wallets buy after launch.

Optional: season helpers, export keys, sweep leftover cash

2 · Fund snipers

Send ETH from treasury → snipers.

Move ETH from your bank to each helper so they can buy. We send a little extra for network fees.

Do this:
  1. Make sure the budget was saved
  2. Preview → check the list looks right
  3. Send cash → wait until it finishes
  4. If some fail, hit Continue (skips wallets that already have cash)
  5. Go to Go live
Privacy hops
Stronger privacy: ChangeNOW → Across (Base → Robinhood)

Breaks the bank→helper link off-chain. Flow: ChangeNOW (dirty ETH → Base) → Across bridge to each helper on Robinhood. Needs CHANGENOW_API_KEY + a Base-funded PRIVACY_BRIDGE_PK. Live Across only with PRIVACY_FUND_ARM=1.
